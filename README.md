# 🧠 General Knowledge Quiz Application

## Decode Labs Python Programmer Internship - Task 4

### 📌 Overview
This repository contains Task 4 of the Decode Labs Python Programmer Internship.

It is a simple command-line general knowledge quiz application built using Python. The program asks three questions, checks the answers, gives instant feedback, and displays the final score at the end.

---

## 🚀 Features
- Ask multiple general knowledge questions
- Check answers instantly
- Display a final score summary
- Simple command-line interface
