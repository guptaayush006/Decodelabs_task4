# General Knowledge Quiz Application

## Task 4

This repository contains Task 4 of the Decode Labs Python internship.

It is a simple command-line General Knowledge Quiz application built using Python. The program asks a few general knowledge questions, checks the answers, gives immediate feedback, and shows the final score at the end.

## Features
- Ask multiple general knowledge questions
- Check answers instantly
- Display final score
- Simple command-line interface
