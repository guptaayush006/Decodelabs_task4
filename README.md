General Knowledge Quiz

A simple Python-based General Knowledge Quiz created as Task 4 for DecodeLabs.

Repository Name

"decodelabs_task4"

File Name

"task4.py"

Description

This project asks the user three general knowledge questions, checks the answers, and displays the final score.

Features

- 3 General Knowledge Questions
- User Input
- Answer Validation
- Score Calculation
- Final Score Display

Technologies Used

- Python 3

How to Run

1. Clone the repository:
   
   git clone https://github.com/guptaayush006/decodelabs_task4.git

2. Open the project folder:
   
   cd decodelabs_task4

3. Run the program:
   
   python task4.py
